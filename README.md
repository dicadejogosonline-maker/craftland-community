# 🎮 CRAFTLAND COMMUNITY\n\nPlataforma de comunidade Craftland com sistema de mapas, comentários e rankings.\n\n## 🚀 Deploy Rápido (Celular)\n\n### **Opção 1: Render.com (Recomendado)**\n\n1. Vá para: https://render.com\n2. Crie conta com GitHub\n3. Clique em \"New\" → \"Web Service\"\n4. Conecte o repositório: `craftland-community`\n5. Configure:\n   - **Name:** craftland-community\n   - **Runtime:** Node\n   - **Build Command:** `npm install`\n   - **Start Command:** `npm start`\n6. Clique em \"Create Web Service\"\n7. Espere fazer deploy (~2-3 minutos)\n8. Abra o link gerado!\n\n### **Opção 2: Replit.com**\n\n1. Vá para: https://replit.com\n2. Crie conta com GitHub\n3. Clique em \"Import from GitHub\"\n4. Cole: `https://github.com/dicadejogosonline-maker/craftland-community.git`\n5. Clique em \"Run\"\n6.
